Machine Learning e Deep Learning usando Algoritmo de CNN para Predizer Doenças na Planta de Soja Neste projeto, vamos explorar o uso de técnicas de Machine Learning e Deep Learning para predizer doenças na planta de soja. Utilizaremos o algoritmo de Convolutional Neural Networks (CNN) para classificar imagens de plantas de soja em saudáveis ou doentes, de acordo com a presença ou não de sinais de doenças.  A base de dados utilizada será composta por imagens de folhas de soja com diferentes tipos de doenças, tais como ferrugem, mancha parda, crestamento foliar e outras. Serão utilizadas técnicas de pré-processamento de imagens, tais como normalização e redimensionamento, para garantir a qualidade das imagens e melhorar o desempenho do modelo.  Em seguida, será construída uma arquitetura de CNN, que será treinada com as imagens pré-processadas. Para isso, serão utilizadas bibliotecas como Keras e TensorFlow. O objetivo é obter um modelo com alta acurácia na predição de doenças na planta de soja, de modo a auxiliar na identificação precoce de problemas e na prevenção de perdas na produção.  Este projeto tem como objetivo contribuir para o avanço das técnicas de Machine Learning e Deep Learning aplicadas à agricultura, bem como para a melhoria da eficiência e produtividade na produção de soja. O código-fonte estará disponível neste repositório do GitHub, juntamente com a base de dados utilizada e um relatório detalhando os resultados obtidos.
