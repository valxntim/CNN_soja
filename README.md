## Machine Learning e Deep Learning usando Algoritmo de CNN para Predizer Doenças na Planta de Soja
Este é um projeto de Machine Learning e Deep Learning que usa o algoritmo de Convolutional Neural Networks (CNN) para predizer doenças na planta de soja. O objetivo é classificar imagens de folhas de soja em saudáveis ou doentes, de acordo com a presença ou não de sinais de doenças.

## Base de Dados
A base de dados utilizada é composta por imagens de folhas de soja com diferentes tipos de doenças, tais como ferrugem, mancha parda, crestamento foliar e outras. Foram utilizadas técnicas de pré-processamento de imagens, tais como normalização e redimensionamento, para garantir a qualidade das imagens e melhorar o desempenho do modelo.

Treinamento do Modelo
Foi construída uma arquitetura de CNN utilizando as bibliotecas Keras e TensorFlow. O modelo foi treinado com as imagens pré-processadas com o objetivo de obter alta acurácia na predição de doenças na planta de soja, de modo a auxiliar na identificação precoce de problemas e na prevenção de perdas na produção.

Resultados
Os resultados obtidos foram avaliados por meio de métricas de desempenho, como acurácia, precisão e recall. O modelo alcançou uma acurácia de XX%, o que indica um bom desempenho na classificação de imagens de folhas de soja.

Como usar o modelo
Para usar o modelo, basta fornecer uma imagem de uma folha de soja e executar o código fornecido no repositório.

Conclusão
Este projeto tem como objetivo contribuir para o avanço das técnicas de Machine Learning e Deep Learning aplicadas à agricultura, bem como para a melhoria da eficiência e produtividade na produção de soja. O código-fonte está disponível neste repositório do GitHub, juntamente com a base de dados utilizada e um relatório detalhando os resultados obtidos.
